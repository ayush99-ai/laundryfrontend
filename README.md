# Laundry Wallah / Laundry Mart

This is a simple HTML webpage for a laundry service website called **Laundry Wallah / Laundry Mart**.
The page shows information about the services provided, the price list, and a small booking form where users can enter their details.

## Features

* Introduction about the laundry service
* List of services offered
* Price table
* Booking form (Name, Email, Phone)
* Contact information

## Technologies Used

* HTML

## How to Run

1. Download the project files.
2. Open the HTML file in any web browser.
